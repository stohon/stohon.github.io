LogParser 2.2 - Query Builder - Part 2

In the first blog, LogParser 2.2 - Query Builder - Part 1, I gave a brief introduction to Microsoft’s LogParser 2.2 application. Now in Part 2, I’m going to demonstrate building a small application in AngularJS that will provide a user friendly UI for quickly building LogParser queries.  The application is contained completely in one html page using nothing but html, javascript, JQuery and AngularJS. It is fully configurable using json.

Check out the application here: https://stohon.github.io/posts/logparserbuilder.html
